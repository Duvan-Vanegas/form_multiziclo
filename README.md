# Modern Form App (Vite + React + Tailwind CSS)

A modern, responsive, and aesthetically pleasing web application featuring a highly polished contact form.
It uses **Vite 4.4.1** as the build tool to ensure stability and compatibility, as per requirements.

## Features
- **Responsive Design**: Mobile-first architecture adapting to any screen size.
- **Modern UI/UX**: Soft shadows, rounded corners (2xl), smooth state transitions.
- **Form Validation**: Real-time client-side validation logic for required fields and email formats.
- **Loading & Success States**: Submitting states with spinners and an interactive success view.
- **Modular Components**: Reusable UI components for Buttons, Inputs, TextAreas, and Cards.

## Deliverables & Technologies Used
- React (Functional Components with Hooks)
- Tailwind CSS 3 (via PostCSS)
- Lucide React (for modern crisp icons)
- Vite 4.x (Build tool initialized via `create-vite@4.4.1`)

## Installation Steps
1. Make sure you have Node.js installed.
2. Clone or download this project.
3. Open your terminal and navigate to the project directory.
4. Run the following command to install dependencies:
   ```bash
   npm install
   ```

## Running the Application
To start the development server, run:
```bash
npm run dev
```
Then, open your browser and navigate to `http://localhost:5173/` (or the port specified in the terminal).
